### 5. 담당 기능 (백엔드)
#### 백엔드 기능구현
① 데이터수집
- 캠핑장 데이터 : 한국관광공사 고캠핑 정보조회 서비스 API
- 캠핑용품 데이터 : 네이버 쇼핑 API

② 유저페이지
- 회원가입
- 로그인(JWT토큰 사용)
- 마이페이지 정보 수정
- 회원탈퇴

③ 관리자페이지
- 로그인(JWT토큰 사용)
- 회원관리(강제탈퇴 기능)
- 캠핑장관리(상품관리, 예약관리, 후기관리)
- 캠핑용품관리(상품관리-추가,수정,삭제, 주문관리, 후기관리)
